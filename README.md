# 👥 User Data Web Page

A simple and responsive **User Data Web Application** built with **HTML, Bootstrap 5, and JavaScript**. This project fetches user information from a public API and displays it in a clean, responsive table with **Edit** and **Remove** action buttons.

## 🌐 Live Demo

🔗 https://md-rifat-mia.github.io/user-data-html-bootstrap-js/

---

## 📌 Features

- 📋 Display user data in a responsive table
- 🌐 Fetch data using JavaScript Fetch API
- 👤 Show User Name
- 📧 Show Email Address
- 🌍 Show Website
- 🔢 Automatic Serial Number
- ✏️ Edit Button (UI)
- 🗑️ Remove Button (UI)
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight

---

---

## 💻 How It Works

1. The page loads.
2. JavaScript sends a request using the Fetch API.
3. User data is retrieved from a public API.
4. Data is displayed inside a Bootstrap table.
5. Each row includes:
   - Serial Number
   - Name
   - Email
   - Website
   - Edit Button
   - Remove Button
